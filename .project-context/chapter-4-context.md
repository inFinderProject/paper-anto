## 4.1 경쟁 환경 분석 (완료)

### 핵심 인사이트

**ANTO의 블루오션 확인**:
> "서울 1시간 이내 + 자연환경 + 프라이빗 휴식" = ANTO가 유일한 선택지

### 매트릭스 분석 결과

#### 2x3 경쟁 매트릭스

| 위치 구분 | 활동/엔터테인먼트 | 휴식/웰니스 |
|-----------|-------------------|-------------|
| **서울 도심** (30분 이내) | 해당 없음 | 시그니엘, 포시즌스, 조선팰리스, 반얀트리 클럽, 아난티 강남 |
| **서울 근교** (1시간 내외) | 인스파이어, 파라다이스시티 (인천 영종도) | **★ ANTO ★** (북한산) |
| **원거리** (항공/KTX) | 제주신화월드 | 아난티 코브 (부산), WE호텔 제주 |

#### 경쟁 영역별 현황

| 영역 | 현황 | 경쟁 강도 |
|------|------|-----------|
| 도심 + 휴식 | 호캉스 레드오션 (자연 없음) | 🔴 치열 |
| 근교 + 엔터 | 인천 영종도 집중 (카지노, 공연) | 🟡 집중 |
| 원거리 + 휴식 | 시간 투자 필요 | 🟢 분산 |
| **근교 + 휴식** | **ANTO 단독 (경쟁 공백)** | ⚪ 공백 |

### ANTO vs 가장 가까운 경쟁자

| 비교 항목 | 반얀트리/아난티 강남 | 인스파이어/파라다이스 | ANTO |
|-----------|---------------------|----------------------|------|
| **위치** | 도심 (남산, 강남) | 인천 영종도 | 북한산 |
| **자연환경** | 조망만 가능 (남산 뷰) | 없음 (인공 리조트) | **국립공원 자연림** |
| **경험 유형** | 도심 프라이빗 | 엔터테인먼트 | **자연 속 휴식** |
| **VIP 은둔** | 단기만 가능 | 불가능 (노출) | **장기 가능** (콜드플레이 2주) |
| **규모** | 소규모 (50실 이하) | 대규모 (1,000실+) | 중규모 (334실) |

### 경쟁 공백(Gap) 정의

**ANTO만의 영역**:
1. 서울 접근성: 강남 기준 40~50분
2. 자연환경: 북한산 국립공원 자연림 (조망이 아닌 거주)
3. 프라이버시: 회원제 + 자연적 격리 + 대규모 부지
4. 경험: 짧고 깊은 휴식 (Time Poverty 대응)

**경쟁자가 제공하지 못하는 것**:
- 도심 호텔 → 자연환경 없음
- 인천 리조트 → 엔터테인먼트 중심, 휴식 아님
- 원거리 리조트 → 시간 투자 필요

**산출물 형식**: 리포트 + 표 포함
- Executive Summary (핵심 인사이트 3~5개)

- 표(Table) 형식 적극 활용

**언어 및 출처**: 글로벌/영문 리서치 우선 (70%), 한글 보조 (30%)

## 작업 진행 시 유의사항

### CLAUDE.md 지침 준수

```markdown
## 작업 지침

### 보고 원칙
- 작업 계획 및 완료 보고 시 한글로 보고

### 정보·리서치 원칙
- 문서·콘텐츠 작성 시 **현실적인 정보(실제 수치·사실·맥락)**를 기준으로 작성할 것.
- 사실 기반 여부가 애매하거나 기준이 불명확할 경우, 일반 웹 검색 대신 **MCP 기반 도구**를 우선 활용:
  - 예: `mcp__duckduckgo__search`, `mcp__duckduckgo__fetch_content`.
- 외부 정보를 충분히 확인할 수 없을 때는 임의로 단정하지 말고, `가정/가설`, `TODO: 추가 검증 필요` 등으로 명시하여 남길 것.
```

### 슬라이드 제작 원칙

- **1 슬라이드 = 1 메시지** 원칙 유지
- 콘텐츠가 많아도 압축하여 1장으로 제작
- PPT 변환을 고려한 표/시각화 중심 구성
