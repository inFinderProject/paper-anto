# Workflow 01: 1차 목차(가설 구조) 워크플로 가이드

> 역할: 인테이크/브리프를 기반으로 **가설 수준의 1차 목차**를 잡아, 리서치의 방향과 질문을 정의하는 단계입니다.

## 1. 입력과 출력

- 입력:
  - `workflow-00-intake` 단계에서 생성된 브리프 요약
  - 프로젝트 유형/목적/타깃 정보
- 출력:
  - 섹션 4~6개 수준의 **가설 목차**
  - 각 섹션별 “무엇을 더 알아야 하는지” 리서치 질문 초안

## 2. 워크플로 위치

- 00 인테이크 → **01 1차 목차 구성** → 02 리서치 → 03 리서치 요약·브리프 압축 → 04 2차 목차 구성

## 3. 운영 원칙 (초안)

- 1차 목차는 “리서치로 검증·보완할 가설”로 간주한다.
- 각 섹션에는 최소 1개 이상의 리서치 질문을 연결한다.
- 리서치 결과에 따라 2차 목차(Workflow 04)에서 과감하게 합치거나 나누되,  
  03 단계에서 **요약 브리프·리서치 참조 맵**으로 한 번 정리한 뒤 최종 구조를 확정한다.
