# Workflow 05: 콘텐츠·슬라이드 작성 워크플로 가이드

> 역할: 04 단계에서 확정된 목차(섹션 구조)를 기준으로, 실제 PPT 슬라이드/문서용 **본문 콘텐츠 초안**을 작성하는 단계입니다.

## 1. 입력과 출력

- 입력:
  - 04 2차 목차(확정 구조)  
    - 예: `projects/<project-name>/04-outline-second/outline-v2.md`
  - 03 리서치 요약·브리프 압축  
    - `projects/<project-name>/03-synthesis/00-research-synthesis.md`
  - 필요 시 세부 리서치 노트(R1~R6)  
    - `projects/<project-name>/02-research/*.md`

- 출력:
  - 섹션/슬라이드 단위의 콘텐츠 초안 MD  
    - 예: `projects/<project-name>/05-content/섹션1-개요.md`  
          `projects/<project-name>/05-content/섹션2-상권고객.md` 등
  - 각 슬라이드에 들어갈:
    - 슬라이드 제목
    - 한 줄 핵심 메시지
    - 불릿/도표/그래프 설명용 텍스트

## 2. 워크플로 위치

- 00 인테이크 → 01 1차 목차 구성 → 02 리서치 → 03 리서치 요약·브리프 압축 → 04 2차 목차 구성 → **05 콘텐츠·슬라이드 작성**

## 3. 운영 원칙 (초안)

- 05 단계에서는 **목차 구조(섹션/슬라이드 개수)는 웬만하면 바꾸지 않고**,  
  각 슬라이드의 문장·예시·데이터를 채우는 데 집중합니다.
- 각 섹션은 다음 순서로 작업합니다:
  1. 04 단계 outline에서 해당 섹션의 “핵심 질문”과 “포함 내용/아이디어”를 다시 확인
  2. 03-synthesis에서 R코드별 핵심 인사이트 체크리스트를 보며,  
     슬라이드별로 어떤 인사이트를 쓸지 매핑
  3. 실제 PPT에 들어갈 문장을 **한국어 완성 문장 + 짧은 불릿** 형태로 작성
- 슬라이드 텍스트는 이후 레이아웃 패턴/디자인 단계에서 수정될 수 있으므로,  
  **완벽한 문장**보다는 “바로 재가공하기 좋은 초안”을 목표로 합니다.
