# Workflow 04: 2차 목차(확정 구조) 워크플로 가이드

> 역할: 03 단계에서 정리된 요약 브리프와 리서치 참조 맵을 기반으로 **최종에 가깝게 목차를 확정**하고, 이후 레이아웃·콘텐츠 작업에서 구조를 흔들지 않도록 기준을 세우는 단계입니다.

## 1. 입력과 출력

- 입력:
  - 01 1차 목차(가설 구조)
  - 03 리서치 요약·브리프 압축 (`03-synthesis/00-research-synthesis.md`)
- 출력:
  - 확정된 섹션 구조(필요시 섹션 병합/분할/순서 조정)
  - 섹션별 핵심 메시지 1~2줄
  - 이후 레이아웃/콘텐츠 단계에서 사용할 기준 목차

## 2. 워크플로 위치

- 00 인테이크 → 01 1차 목차 구성 → 02 리서치 → 03 리서치 요약·브리프 압축 → **04 2차 목차 구성**

## 3. 운영 원칙 (초안)

- 2차 목차 이후에는 **섹션 구조를 웬만하면 변경하지 않는다.**
- 섹션 안에서의 슬라이드 추가/삭제로 유연성을 확보한다.
- 구조 결정이 끝나면, 레이아웃 패턴 선택과 콘텐츠 작성으로 곧바로 이어간다.
- 2차 목차 결과물은 프로젝트별 `04-outline-second/outline-v2.md`에 정리하며,  
  이후 단계(레이아웃 패턴 매핑, 슬라이드/콘텐츠 작성)는 이 파일을 기준으로 진행한다.
