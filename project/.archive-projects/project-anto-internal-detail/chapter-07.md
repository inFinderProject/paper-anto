# 챕터 7
조직별 실행 체계

## 7.1 부서별 핵심 미션
- 세일즈: 멤버십 분양률 24%→60% 달성, 월 5건 1:1 계약, 채널별 리드→투어→계약 전환 극대화.
- 마케팅: VIP 인지도 70%/대중 50% 목표, 채널 믹스 최적화(골프/계열/디지털/PR), 콘텐츠·투어 패스 운영.
- 운영: OCC 66→75, ADR 40→45, NPS 70, 업셀/부대수익 40→50% 달성; 체크인·룸서비스·NPS/VOC SOP 실행.
- F&B/웰니스: 시그니처 F&B 리뉴얼, 웰니스 6Pillar 프로그램 운영, 원가·마진 관리, 후기/NPS 개선.
- 디지털/IT: PMS/RMS/예약/CRM 연동, SLA 알림(응답/투어/계약), 예약 성공률/오류율 모니터링.
- 재무: 분양/운영 KPI 재무 반영, CAC·ROI·CapEx 집행 통제, 시나리오별 비용 관리.
- HR/교육: 식음·프런트 충원, 서비스·업셀 트레이닝, 교차교육·핵심 인력 백업.
- CX/데이터: 리드 태깅·대시보드 운영, NPS·VOC 수집/분석, 재방문/추천률 추적.

## 7.2 KPI 대시보드
- 분양: 리드/투어/계약 전환율, CAC(목표: 4,650만/실 이하), 월 계약 수, 잔여 물량.
- 운영: OCC(목표 75), ADR(45만원), RevPAR, 부대수익 비중(50%), 인건비율(43% 이하).
- 경험: NPS(70), VOC 처리 TAT, 재방문/추천률.
- 채널: 채널별 리드·계약·ROI, 제휴별 매출 기여, 캠페인별 성과.
- 디지털: 온라인 예약 비중, 예약 성공률/오류율, SLA 준수율(응답/투어/계약).
- 재무: 투자 집행(브랜드/마케팅/운영 60.2억), ROI(1,220%) 진행률, CapEx 집행 vs 예산.

## 7.3 실행 타임라인
- Q1: 분양가/혜택 확정·공표, 인센티브 3단계 런칭, 리드 대시보드 오픈, 투어 SLA 시행, 디지털 최소 기능 배포, 인력 충원/교육.
- Q2: 채널·캠페인 최적화(리드→계약 개선), 패키지 SKU 고도화(가족/실버/MICE), NPS/VOC 루프 정착, F&B/웰니스 리뉴얼 적용.
- Q3: 분양률 중간 점검, 법인/외국인 비중 확대, 디지털 자동화(SLA 알림, 데이터 통합), 업셀/부대수익 강화.
- Q4: 연말 총괄 리프레시, 가격/인센티브 조정, 내년도 CapEx·마케팅 예산 수립, 성과 리뷰/보상·교육 계획.

## 7.4 협업 체계
- 리드·채널: 마케팅/세일즈 공동 소유, CRM·데이터 지원, 재무 모니터링.
- 패키지/상품: 운영(F&B/웰니스) 주도, 세일즈·마케팅 협업, 재무 검토.
- 디지털: IT 리드, 세일즈/운영 협의, 경영진 승인, 데이터/CRM 연동.
- KPI/대시보드: 데이터 팀 관리, 세일즈·운영 입력, 경영진 모니터링.
- 리스크/비상: 분양·OCC 쇼크, 디지털 장애, 인력 공백 시 RACI 기반 비상 태스크포스 가동(세일즈/운영/IT/재무).
