- 인수/가치: 300억 인수 → 6,000억 가치, 차익 ~2,000억 (일관).  
- 투자/ROI: 총 60.2억(브랜드 16.7/마케팅 37.2/운영 8.0) → 효과 734억, ROI 1,220% (내·외부 모두 동일).  
- 분양: 현재 24% → 목표 60%(3년), 추가 80실 400억, 총 분양가 가설 5,900~6,000억 (내·외부 동일).  
- 운영: OCC 66→75, ADR 40→45, 부대수익 40→50%, 영업이익 △171→+100, NPS 70 (내부 슬라이드에 반영, 외부에서는 강조 안 함).  
- 마케팅: CAC 4,650만/실, 마케팅 ROI 1,075%, 인센티브 3단계(5%/2~3%/법인5%).  
- 확인 필요: 타입별 분양가·혜택·월별 목표·금융 옵션, 디지털 롤아웃 일정/벤더, 프로그램 캘린더·NPS 데이터, 교통 실측/뷰 컷, CTA SLA.
