# PPT 템플릿 구조 (슬라이드 타입 목록)

## 공통 마스터
- 커버: 히어로 이미지+오버레이, 타이틀/서브/CTA.
- 섹션 구분: 단색 배경, 큰 번호/제목.
- 엔딩/CTA: 라이트 배경, 한줄 메시지+CTA 버튼/담당자.

## 콘텐츠 타입
- 강조형 highlight-01: 헤드라인 + KPI 카드 3~4.
- 히어로 visual-hero-01/02: 이미지 풀블리드 + 텍스트.
- 기능 그리드 feature-grid-02/03: 2x2 또는 2x3 카드(아이콘+텍스트).
- 데이터/표 table-02/03: 헤더 강조 표, 2~3컬럼.
- 프로세스 process-01/02: 3~4단계 노드+화살표.
- 타임라인 timeline-01: 3단계 막대/노드.
- 스윔레인 swimlane-01: 채널별 행×단계별 열.
- 카드 card-04: 세그먼트/패키지별 카드형.
- Q&A qa-01: 질문/답변 카드 2~4개.
- 증빙 proof-02: 3~4 포인트 + 비주얼.

## 내부용 추천 조합
- Exec: highlight-01
- 입지: visual-hero-02
- 재무/분양: data-02 + timeline-01
- 상품/콘텐츠: feature-grid-03
- 영업/채널: swimlane-01
- 운영/디지털: process-02
- 리스크: table-02

## 대외용 추천 조합
- 오프닝: visual-hero-01
- 왜 ANTO: problem-solution-02
- 가치: feature-grid-02
- 패키지: card-04
- 증빙: proof-02
- 가격·인센티브: table-03
- 이의제기: qa-01
- CTA: process-01
