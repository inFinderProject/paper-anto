# ANTO 뉴스 기반 인테이크 브리프 - 대외용 (Workflow 00)

## 1) 컨텍스트
- 범위: `news/`의 ANTO 관련 2025년 9~10월 보도 49건 검토.
- 목적: 대외( VIP 고객·파트너·언론)용 메시지·목차·스타일 방향 확정 후 1차 목차(Workflow 01)로 연계.

## 2) 대상·관심사
- VIP 고객, 파트너사·협력기관, 언론.
- 관심사: 프라이빗 멤버십 희소성·프리미엄 가치, 브랜드 철학·지속 가능 비전, VIP 라이프스타일 맞춤 제안, 투자/구매 의사결정 근거.

## 3) 목표·성공 기준
- 서울 유일 입지·희소성 기반 프리미엄 인지 제고.
- 멤버십 리드 확보(분양률 24%→가속) 및 전략 파트너십/MICE 파이프라인 확대.
- 브랜드 감성+투자가치를 동시 설득, 투어/상담 CTA 전환.

## 4) 뉴스 기반 핵심 메시지·팩트
- 입지·희소성: 북한산국립공원 자락 2.4만평, 도심 40~50분 ‘단 한 곳’ 리조트 `news/31-News.md`, `news/52-News.md`.
- 프라이빗 멤버십: 224실 한정, 6타입(67~156평) 발코니/히노키/인피니티풀 독점, 분양률 24% 진행 중 `news/02-News.md`, `news/25-News.md`.
- 브랜드 스토리: “그 땅에서의 편안한 삶 / Where luxury meets tranquility”, 천연 소재·600년 나무 보존, 콜드플레이 숙박 사례, 시그니처 F&B `news/31-News.md`, `news/39-News.md`, `news/72-News.md`.
- 경험 가치: 인피니티풀·온수풀 4곳, 요가·명상·숲 테라피·루프톱 천체 프로그램, 8인 패밀리 스위트, 앱 기반 디지털 서비스 `news/02-News.md`, `news/72-News.md`, `news/39-News.md`.
- 투자/가치 포인트: 300억 투자로 6,000억 자산 확보, 분양 한정성·자산가치 상승 스토리 `news/52-News.md`, `news/87-News.md`.

## 5) 대외용 문서 목차 가이드 (여백+이미지 중심)
1. 서울 도심 40분·국립공원 파노라마(지도·뷰)
2. 브랜드 스토리·지속가능 콘셉트(천연 소재·보존 사례, 슬로건)
3. 프라이빗 멤버십 가치(224실 한정, 6타입, 인피니티풀/발코니/히노키 독점)
4. 경험 시나리오: 웰니스·미식·패밀리·MICE, 콜드플레이 사례
5. 투자/구매 근거: 한정 분양·가격대·자산가치 포인트, 파트너십 기회
6. 콜투액션: 멤버십 상담/투어 예약, 공동 사업/행사 제안

## 6) 스토리텔링/톤 제안
- 여백과 이미지 우선, 감성 카피 + 짧은 팩트.
- 희소성(“서울 유일·단 하나”)과 프라이버시 강조, 체험 중심 비주얼(뷰·웰니스·F&B).
- CTA: 투어 예약/상담, 파트너십 미팅, MICE/웨딩 문의.
