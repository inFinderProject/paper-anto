# 1차 목차 (대외용) - ANTO 브랜드·멤버십 제안

1. 서울 단 한 곳: 40분·국립공원 파노라마 입지  
   - 지도/뷰 비주얼, 희소성·접근성 스토리.
2. 브랜드 스토리·지속가능 콘셉트  
   - “Where luxury meets tranquility”, 천연 소재·600년 나무 보존, 감성 카피.
3. 프라이빗 멤버십 가치  
   - 224실 한정, 6타입(67~156평) 발코니/히노키/인피니티풀 독점, 분양 진행률·한정성.
4. 경험 시나리오 (웰니스·미식·패밀리·MICE)  
   - 요가/명상·숲 테라피·루프톱 천체, 시그니처 F&B, 8인 스위트·패밀리/기업 패키지.
5. 투자/구매 근거 및 콜투액션  
   - 자산 가치 스토리(300억→6,000억), 분양가/혜택, 파트너십·MICE 제안, 투어/상담 CTA.

- 입지·희소성: 정확한 이동 시간(차량/대중교통), 경쟁 리조트 대비 차별 포인트 시각자료 확보?
- 브랜드·지속가능: 소재·친환경 설계 인증 여부, 브랜드 슬로건/톤앤매너 레퍼런스 필요?
- 멤버십: 타입별 USP·가격대·혜택(서비스·예약 우선권), 분양 잔여 물량·타임라인?
- 경험 시나리오: 프로그램 캘린더(계절/주간), MICE/웨딩 수용 규모·패키지 가격, 대표 후기·사례(콜드플레이 등)?
- 투자/CTA: 분양 특전·금융 옵션, 파트너십 가능 영역(공동 프로모션/콘텐츠), 상담·투어 예약 플로우/SLAs?
