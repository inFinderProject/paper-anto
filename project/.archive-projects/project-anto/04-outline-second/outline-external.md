# 2차 목차 (대외용) - ANTO 브랜드·멤버십 제안

## 섹션 구조 (슬라이드 단위 제안)
1. **Only-One Location**  
   - 서울 40~50분·국립공원 파노라마, 복제 불가 희소성(뷰/지도 시각).  
2. **Brand Story & Sustainability**  
   - “Where luxury meets tranquility”, 천연 소재·600년 나무 보존, Pure Refinement 웰니스 철학.  
3. **Private Membership Value**  
   - 224실 한정·6타입(67~156평) 발코니/히노키/인피니티풀 독점, 분양률 24% 진행·한정성.  
4. **Experience Scenarios**  
   - 웰니스(요가/명상/숲 테라피/루프톱 천체)·미식(시그니처 F&B)·패밀리(8인 스위트)·MICE/웨딩 패키지.  
5. **Investment/Decision Drivers**  
   - 300억→6,000억 가치 스토리, 3년 내 완판 목표, 단계별 인센티브(얼리버드 5% 할인·스파 혜택 등), 희소성·가치 상승 포인트.  
6. **Call to Action**  
   - 투어/상담 예약 프로세스·SLA, 파트너십/공동 프로모션 제안, 리드 처리 흐름.

## 슬라이드별 필요 데이터/증빙 (R코드 참조)
- Location: R1 뉴스(40~50분·국립공원 뷰), 교통 실측·지도/뷰 컷 필요(오픈).  
- Brand: R3 포지셔닝(슬로건·철학), R1/R2 스토리(콜드플레이·600년 나무), 친환경 인증 여부(오픈).  
- Membership: R1/R4(분양률 24%, 6타입 특장점, Tier 타깃), 타입별 가격·혜택·잔여 물량·타임라인(오픈).  
- Experience: R1/R5(시설·프로그램), 프로그램/패키지 가격·수용 규모·후기/NPS(오픈).  
- Investment: R6(300억→6,000억 가치, ROI), R4 인센티브 단계, 최신 금융 옵션(오픈).  
- CTA: 상담/투어 예약 플로우·SLA, 파트너십 제안 영역(오픈).

## 오픈 이슈 정리 (채우기 전)
- 교통 실측·지도/뷰 시각 자료.  
- 멤버십 타입별 가격·혜택·잔여 물량·예약 프로세스/SLA.  
- 프로그램·패키지(웰니스/패밀리/MICE/웨딩) 구성·가격·후기.  
- 친환경 인증/수상, 브랜드 비주얼 톤앤매너.  
- 최신 분양 특전·금융 옵션, CTA 플로우·리드 처리 SLA, 파트너십 가능 영역.
