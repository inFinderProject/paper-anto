# 1차 목차 (대내용) - ANTO 인수·통합 전략

## 섹션 가설 (4~6개)
1. 인수 배경·자산 가치 재평가  
   - 300억 인수로 6,000억 가치 확보, 서울 유일 국립공원 입지·희소성 정리.
2. 통합 로드맵: 운영 안정화 → 분양 가속 → 디지털/서비스 고도화  
   - 시설 리프레시·어메니티 교체, 분양 리드 파이프라인, 앱/디지털 체크인.
3. 재무·분양 KPI와 성장 드라이버  
   - 분양률 24%→60%(3년), 분양 2,000억+/매출 500억+, 성장률 200%, 흑자 전환.
4. 상품·콘텐츠 강화 (멤버십/웰니스/F&B)  
   - 6개 멤버십 타입, 웰니스·루프톱·패밀리 프로그램, 시그니처 F&B.
5. 영업·채널·전사 협업 체계  
   - 더플라자 네트워크, 인바운드 여행사, MICE/웨딩, 계열 CRM·F&B(푸드테크·아워홈).
6. 리스크·거버넌스·성과 관리  
   - 분기 KPI, 책임(RACI), 리스크·대응(분양 속도, 인지/브랜드, 시설 유지보수).

## 섹션별 리서치 질문 (검증·보완 포인트)
- 인수 배경·가치: 실제 부채 상계 후 자산가치·ROI, 유사 입지/경쟁 부재 사례 근거?
- 통합 로드맵: 시설 보수 범위·CapEx/TIMELINE, 디지털 체크인/앱 현 수준과 롤아웃 계획?
- 재무·분양: 분양가 밴드·할인 정책, 월별 분양 속도 목표, 객실점유율·ADR 가설?
- 상품·콘텐츠: 웰니스/패밀리/MICE별 프로그램 캘린더, 시그니처 F&B 라인업·가격, NPS/리뷰 인사이트?
- 영업·채널: 더플라자·계열 CRM 교차판매 계획, 인바운드 여행사·MICE 타깃 리스트, 외국인 비중 목표?
- 리스크·거버넌스: 분양 미달/시황 악화 시 플랜 B, 시설 유지보수 비용 추정, 브랜드 인지도 제고 지표?
