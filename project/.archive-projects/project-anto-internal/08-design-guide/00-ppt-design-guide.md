# PPT 디자인 가이드 (ANTO 인수/세일즈 덱 공통)

## 1. 기본 원칙
- 가독성 우선: 16:9, 회의실 프로젝터 기준, 본문 18~20pt, 라인간격 1.4~1.5.
- 여백/호흡: 상단 제목 영역 확보, 본문 그리드 12열(여백 좌우 48px 내외) 활용.
- 컬러 대비: 다크 텍스트 + 라이트 배경 기본, 포인트 컬러로 액센트. 데이터 강조만 포인트 사용.

## 2. 컬러 팔레트
- 기본 배경: #F8F9FA (라이트 그레이), 보조 배경 블록: #EEF1F4.
- 텍스트: 헤드라인 #0F172A, 본문 #1F2937, 캡션 #4B5563.
- 포인트: 브랜드 포인트(딥 그린) #0F4D3A, 서브 포인트(샌드) #C6A982, 상태 색상(성장/긍정) #16A34A, 경고 #F97316.
- 데이터 색: 메인 #0F4D3A, 보조 #C6A982, 대비용 #2563EB. 그래프는 2~3색 내로 제한.

## 3. 타이포그래피
- 제목/헤드라인: Pretendard SemiBold 28~32pt.
- 본문: Pretendard Regular/Semibold 18~20pt.
- 캡션/주석: 14~16pt.
- 숫자/데이터: 동일 폰트 Semibold, 단위는 얇게(Regular).
- 줄간격: 제목 1.2, 본문 1.4~1.5, 표/리스트 1.3.

## 4. 그리드·레이아웃 규칙
- 16:9 캔버스, 좌우 여백 48px, 상단 제목 72~96px 영역 확보.
- 카드/그리드: 최대 4열(모바일 고려 시 2열 전환 가능), 아이콘 상단 + 텍스트 아래.
- 표: 행 높이 최소 32px, 헤더 Bold, 홀수/짝수 줄 그레이 라이트 스트라이프.
- 타임라인/프로세스: 좌→우 혹은 상→하 일관, 단계 수 3~5.

## 5. 아이콘·사진 스타일
- 아이콘: 단색 라인/단색 면 혼합(라인 두께 2px), 포인트 컬러 한정 사용.
- 사진: 밝은 톤, 자연·공간·뷰 강조. 비네팅/필터 최소. 동일 톤(차가운/따뜻한) 혼용 자제.
- 비주얼 우선 슬라이드(히어로): 풀블리드+어둡게 오버레이 20~30% 후 텍스트 올리기.

## 6. 표·그래프 스타일
- 표: 상단 헤더 바탕 #EEF1F4, 텍스트 #0F172A, 강조 셀 배경 포인트 라이트(#E5ECE8). 테두리 라인 #E5E7EB.
- 그래프: 막대/선 2~3색, 데이터 라벨는 필요한 곳만. 단위는 y축 혹은 상단에 명확히.
- 인포박스: 숫자 강조 박스 배경 #E5ECE8, 텍스트 #0F172A, 포인트 수치에만 #0F4D3A.

## 7. 슬라이드 타입별 템플릿 가이드
- 히어로(visual-hero-01/02): 상단 좌/우 텍스트 + 배경 이미지 풀블리드, 오버레이 20~30%.
- 강조형(highlight-01): 상단 헤드라인, 하단 3~4 KPI 카드(아이콘+숫자).
- 데이터/표(table-03): 2~3컬럼 표, 헤더 강조, 빈칸은 대시(-) 처리.
- 그리드(feature-grid-02/03): 2x2 또는 2x3 카드, 아이콘+한줄 메시지+짧은 본문.
- 프로세스(process-01/02): 3~4단계, 원 또는 직사각형 노드+화살표, 단계별 한줄+서브 텍스트.
- 타임라인(timeline-01): 3단계 인센티브/로드맵, 좌→우 막대형.
- 스윔레인(swimlane-01): 채널별 행, 리드→투어→계약 단계 열, KPI 캡션.
- Q&A(qa-01): 질문 박스+답변 박스 카드형, 2~4개.

## 8. 챕터/커버/엔딩
- 커버: 배경 이미지(뷰) + 반투명 오버레이 + 타이틀/서브/CTA.
- 섹션 구분: 단색 배경(포인트 혹은 라이트 그린), 큰 번호/제목.
- 엔딩/CTA: 단색 라이트 배경, 한줄 메시지+CTA 버튼/담당자 연락처.

## 9. 내부/외부 차별화 포인트
- 내부: 데이터/표/프로세스 비중 높음, 포인트 컬러 사용 최소, 텍스트 대비 강화.
- 외부: 비주얼(뷰/공간) 강조, 카드/히어로 슬라이드 활용, CTA 명확히 배치.
