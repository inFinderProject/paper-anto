# Chapter 4 레이아웃 와이어프레임 가이드  

> **대상 챕터**: `Chapter 4. 시장 분석 및 포지셔닝`  
> **기준 콘텐츠**: `project/05-content/chapter-04-draft-01.md`  
> **슬라이드 크기**: 4:3 Landscape (예: 1024×768px, 단위는 상대 비율 기준)  
> **레이아웃 철학**:  
> - `ANTO_경영 보고서_내부용_v4.pdf`의 페이지 구조 → **제목 아래 설명글, 그 아래 본문 블록**  
> - `ANTO_디자인 리포트 가이드.pdf`의 Grid System(풀 / 2분할 / 3분할 / 비대칭 / 중앙집중 / 4분할)을 활용  
> - **메시지 전달 우선**: 한 슬라이드에 1~2 메시지, 숫자는 필요한 최소만

## 1. 공통 사양 (4:3 ANTO 슬라이드)

- **캔버스 크기(가정)**: 1024 × 768 px  
- **여백(추천)**: 상하좌우 48 px  
- **영역 분할**
  - 상단 헤더 바: 높이 약 64 px (`header-bar`)
  - 본문 영역 상단: 섹션 제목 + 설명글(제목 아래 2~3줄)  
  - 본문 영역 하단: 콘텐츠 블록(표/그래프/카드 등)

### 공통 레이아웃 컴포넌트

| 컴포넌트 | 설명 |
|----------|------|
| `header-bar` | 상단에 챕터/섹션 정보, 로고, 페이지 번호(선택) |
| `title-box` | 슬라이드 제목(좌측 정렬 또는 중앙 정렬) |
| `subtitle-box` | 제목 바로 아래 1~2줄 설명 텍스트 |
| `content-area` | 실질적 콘텐츠가 들어가는 영역(그리드 패턴 적용) |
| `note-box` | 시사점·메모(슬라이드 하단 또는 우측) |
