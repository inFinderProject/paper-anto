## 1. 공통 사양 (4:3 ANTO 슬라이드)

- **캔버스 크기(가정)**: 1024 × 768 px  
- **여백(추천)**: 상·하·좌·우 48 px  
- **영역 분할**
  - 상단 헤더 바: 높이 약 64 px (`header-bar`)
  - 본문 상단: 슬라이드 제목(`title-box`) + 설명글(`subtitle-box`)  
  - 본문 하단: 콘텐츠 영역(`content-area`) – 부서 카드, KPI 표, 타임라인, 스윔레인 등

### 공통 레이아웃 컴포넌트

| 컴포넌트 | 설명 |
|----------|------|
| `header-bar` | 상단에 챕터/섹션 정보, 로고, 페이지 번호(선택) |
| `title-box` | 슬라이드 제목(좌측 정렬 추천) |
| `subtitle-box` | 제목 아래 1~2줄 설명 텍스트 |
| `content-area` | 조직/미션/KPI/협업 구조를 배치하는 영역 |
| `note-box` | 보조 설명(책임자, 가정, 연결 관계 등) |
