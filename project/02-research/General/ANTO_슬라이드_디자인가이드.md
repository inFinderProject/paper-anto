## 1. 개요

### 1.1 문서 목적
본 가이드는 ANTO 최종 보고서(총 152장)를 PowerPoint로 제작할 때 일관된 디자인과 브랜드 아이덴티티를 유지하기 위한 실무 참고 자료입니다.

### 1.2 ANTO 브랜드 철학
- **브랜드명**: 安土 (안토) - "편안할 안(安)" + "흙 토(土)"
- **슬로건**: "Where luxury meets tranquility"
- **3P 컨셉**: Personal, Private, Purpose
- **핵심 가치**: 자연과의 공존, 정화와 균형, 프리미엄 힐링

### 1.3 전체 슬라이드 구성 (152장)

| 섹션 | 슬라이드 수 | 주요 내용 |
|------|------------|----------|
| Executive Summary | 12장 | 핵심 메시지, 주요 현안, 향후 방향 |
| I. 제안 개요 및 전략 | 18장 | 브랜드 아이덴티티, 비전, 전략 |
| II. 운영 현황 | 22장 | 조직, 시설, 재무 운영 현황 |
| III. 시장 분석 및 경쟁 현황 | 13장 | 시장 트렌드, 경쟁사 분석, SWOT |
| IV. 마케팅 및 영업 전략 | 23장 | 타겟 고객, 회원권 판매, 마케팅 |
| V. 주요 플랜 및 이슈 | 20장 | 단기 과제, 리스크 관리 |
| VI. 향후 액션 과제 | 26장 | 실행 계획, 부서별 KPI, 로드맵 |
| VII. 재무 현황 및 전망 | 18장 | 손익 구조, 2026년 흑자 전환 계획 |
| **합계** | **152장** | - |

## 3. 타이포그래피

### 3.1 한글 폰트

#### 제목 (Headings)
- **폰트**: Noto Sans KR Bold 또는 맑은 고딕 Bold
- **사이즈**:
  - 슬라이드 제목: 40-48pt
  - 섹션 구분 타이틀: 60-72pt (중앙 정렬)
  - 중제목 (Sub-title): 28-32pt
  - 소제목 (Minor title): 20-24pt
- **색상**: 다크 그린 (#3E4A3D)
- **행간**: 1.2배

#### 본문 (Body Text)
- **폰트**: Noto Sans KR Regular 또는 맑은 고딕 Regular
- **사이즈**: 16-20pt
- **색상**: 차콜 그레이 (#4A4A4A)
- **행간**: 1.5배

#### 강조 (Emphasis)
- **폰트**: Noto Sans KR Bold 또는 맑은 고딕 Bold
- **사이즈**: 본문 +2pt (18-22pt)
- **색상**: 우드 브라운 (#8B7355) 또는 다크 그린
- **활용**: 핵심 수치, 중요 키워드

### 3.2 영문 폰트

#### 제목
- **폰트**: Montserrat Bold 또는 Arial Bold
- **사이즈**: 한글 제목과 동일 (40-48pt)
- **활용**: "ANTO", "KPI", "ROI" 등 브랜드명/약어

#### 본문
- **폰트**: Montserrat Regular 또는 Arial Regular
- **사이즈**: 16-20pt
- **활용**: 슬로건, 영문 설명

### 3.3 숫자 폰트
- **폰트**: Arial Bold 또는 Montserrat Bold
- **사이즈**:
  - 핵심 지표: 32-48pt (대형 박스 내)
  - 표 내 숫자: 18-20pt
  - 일반 숫자: 16pt
- **색상**:
  - 긍정: #28A745 (성공 그린)
  - 부정: #DC3545 (위험 레드)
  - 중립: #4A4A4A (차콜 그레이)

### 3.4 폰트 크기 가이드 요약

| 요소 | 사이즈 | 폰트 | 색상 |
|------|--------|------|------|
| 섹션 타이틀 | 60-72pt | Noto Sans KR Bold | 다크 그린 |
| 슬라이드 제목 | 40-48pt | Noto Sans KR Bold | 다크 그린 |
| 중제목 | 28-32pt | Noto Sans KR Bold | 다크 그린 |
| 소제목 | 20-24pt | Noto Sans KR Bold | 우드 브라운 |
| 본문 | 16-20pt | Noto Sans KR Regular | 차콜 그레이 |
| 핵심 수치 | 32-48pt | Arial Bold | 상황별 |
| 표 수치 | 18-20pt | Arial Regular | 차콜 그레이 |
| 출처/각주 | 12-14pt | Noto Sans KR Regular | 라이트 그레이 |

### 3.5 행간 및 자간
- **행간 (Line Spacing)**:
  - 제목: 1.0-1.2배
  - 본문: 1.5배
  - 표: 1.2배
- **자간 (Letter Spacing)**:
  - 기본값 (0)
  - 제목만 +5% (선택 사항)

## 4. 레이아웃 템플릿 (12가지)

### 4.1 Section Divider (섹션 구분)
**용도**: 각 챕터 첫 슬라이드

**구조**:
```
┌─────────────────────────────────┐
│                                 │
│         [중앙]                  │
│    I. 제안 개요 및 전략          │
│                                 │
│         [하단]                  │
│  브랜드 · 전략 · 히스토리        │
│                                 │
└─────────────────────────────────┘
```

**디자인 요소**:
- 배경: 베이지 (#D4C5B0) 또는 ANTO 로고 워터마크 (투명도 10%)
- 제목: 중앙 정렬, 60-72pt, 다크 그린
- 하단 설명: 중앙 정렬, 24pt, 우드 브라운

**사용 예시**: Executive Summary, I~VII 각 섹션 첫 슬라이드

---

### 4.2 Dashboard (KPI 박스)
**용도**: 핵심 지표를 한눈에 보여줄 때

**구조**:
```
┌─────────────┬─────────────┬─────────────┐
│ 회원권 분양률│ 객실 점유율  │ 월평균 매출  │
│             │             │             │
│    24%      │    66%      │  27.7억원   │
│             │             │             │
│ 목표: 60%   │ 목표: 75%   │ 목표: 2배   │
└─────────────┴─────────────┴─────────────┘
```

**디자인 요소**:
- 박스: 2x2 또는 3열 그리드
- 현재 값: 32-48pt, Bold, 중앙 정렬
- 목표 값: 16-18pt, 파란색 (#007BFF)
- 테두리: 1pt, 라이트 그레이
- 배경: 흰색 또는 베이지 10%

**사용 예시**: Executive Summary 슬라이드 3, II. 운영 현황 슬라이드 2

### 4.3 Data Table (데이터 표)
**용도**: 정량 데이터 비교

**구조**:
```
┌────────────────────────────────────────┐
│ 부서    정원  재직  결원  결원율       │
│─────────────────────────────────────  │
│ 객실팀   51    49     4    7.8%       │
│ 식음팀   48    37    13   27.1% ⚠️    │
│ 조리팀   56    52     5    8.9%       │
│─────────────────────────────────────  │
│ 합계    210   195    15    7.1%       │
└────────────────────────────────────────┘
```

**디자인 요소**:
- 헤더: 다크 그린 배경 (#3E4A3D), 흰색 텍스트
- 홀수 행: 흰색 배경
- 짝수 행: 라이트 그레이 배경 (#F5F5F5)
- 테두리: 1pt, 차콜 그레이
- 숫자: Arial, 우측 정렬
- 경고 행: 빨간색 텍스트 또는 배경

**사용 예시**: II. 운영 현황 슬라이드 4, 11, 16

### 4.4 Comparison (좌우 비교)
**용도**: Before/After, 현재/목표 비교

**구조**:
```
┌──────────────────┬──────────────────┐
│ 2025년 (현재)    │ 2026년 (목표)    │
│                  │                  │
│ OCC: 66%         │ OCC: 75%         │
│ ADR: 40만원      │ ADR: 45만원      │
│                  │                  │
│ 결원율: 7.1%     │ 결원율: 0%       │
└──────────────────┴──────────────────┘
```

**디자인 요소**:
- 2개 컬럼 (50:50 또는 40:60 비율)
- 중앙에 화살표 (→) 배치
- 현재: 회색 배경, 파란색 텍스트
- 목표: 흰색 배경, 녹색 텍스트
- 개선 항목: 녹색 체크마크 (✅)

**사용 예시**: Executive Summary 슬라이드 9, VI. 향후 액션 과제 슬라이드 7

### 4.5 Timeline / Roadmap (타임라인)
**용도**: 시간 순서, 단계별 계획

**구조**:
```
2025 Q4     2026 Q1     2026 Q2     2026 Q3     2026 Q4
   ↓           ↓           ↓           ↓           ↓
 10실        25실        20실        15실        10실
 분양        분양        분양        분양        분양
```

**디자인 요소**:
- 수평 타임라인 (5-6개 단계)
- 화살표 연결 (→ 또는 ↓)
- 각 단계: 박스 + 제목 + 세부 내용
- 진행 단계: 색상 코딩 (완료=녹색, 진행중=파란색, 예정=회색)
- 마일스톤 강조 (별 아이콘 ⭐)

**사용 예시**: Executive Summary 슬라이드 8, IV. 마케팅 슬라이드 21

### 4.6 Matrix (2x2, 3x3 매트릭스)
**용도**: SWOT, 리스크 분석, 포지셔닝

**구조**:
```
           영향도 High
               ↑
    ┌──────────┼──────────┐
    │ 2. 브랜드 │ 1. 재무  │
High│ 인지도    │ 건전성   │
발생│──────────┼──────────│
확률│ 4. 인력  │ 3. 경기  │
    │ 부족     │ 침체     │
    └──────────┼──────────┘
               ↓
           Low
```

**디자인 요소**:
- 2축 (X, Y) 화살표로 축 표시
- 4사분면 색상 코딩:
  - 좌상단 (High/High): 빨간색 배경 (위험)
  - 우상단 (High/Low): 주황색 배경 (주의)
  - 좌하단 (Low/High): 노란색 배경 (관찰)
  - 우하단 (Low/Low): 녹색 배경 (안전)
- 항목 번호 표시 (1, 2, 3, 4)

**사용 예시**: Executive Summary 슬라이드 10, III. 시장 분석 슬라이드 12, V. 주요 플랜 슬라이드 15

### 4.7 Process Flow (프로세스 흐름)
**용도**: 단계별 절차, 업무 프로세스

**구조**:
```
[1. 접수] → [2. 배정] → [3. 응답] → [4. 해결] → [5. 피드백]
   ↓            ↓           ↓           ↓            ↓
 당일        2시간       24시간       5일          1일
```

**디자인 요소**:
- 5-7단계 수평 배치
- 각 단계: 둥근 직사각형 박스
- 화살표 연결 (→)
- 단계 번호 표시 (1, 2, 3...)
- 기한/책임자 하단 배치

**사용 예시**: V. 주요 플랜 슬라이드 10, VI. 향후 액션 과제 슬라이드 16

### 4.8 SWOT Analysis (SWOT 분석)
**용도**: 강점, 약점, 기회, 위협 분석

**구조**:
```
┌───────────────────┬───────────────────┐
│ Strengths (강점)  │ Weaknesses (약점) │
│ ✅ 유일무이한 입지 │ ⚠️ 낮은 인지도    │
│ ✅ 회원제 모델     │ ⚠️ 높은 결원율    │
└───────────────────┴───────────────────┘
┌───────────────────┬───────────────────┐
│ Opportunities     │ Threats (위협)    │
│ 🔷 웰니스 성장     │ 🔸 경쟁 심화      │
│ 🔷 VIP 니즈 증가  │ 🔸 경기 침체      │
└───────────────────┴───────────────────┘
```

**디자인 요소**:
- 4사분면 (2x2)
- 강점 (Strengths): 녹색 배경 (#6FCF97)
- 약점 (Weaknesses): 빨간색 배경 (#FFE6E6)
- 기회 (Opportunities): 파란색 배경 (#E3F2FD)
- 위협 (Threats): 주황색 배경 (#FFF3E0)
- 아이콘 활용 (✅, ⚠️, 🔷, 🔸)

**사용 예시**: II. 운영 현황 슬라이드 18, III. 시장 분석 슬라이드 12

### 4.9 3-Column Grid (3열 그리드)
**용도**: 3가지 전략, 세그먼트, 옵션 비교

## 5. 데이터 시각화 가이드

### 5.1 표 (Table) 스타일

#### 표준 표 형식
```
┌──────────────────────────────────────┐
│ 항목      2025년   2026년   증감률   │ ← 헤더 행
│──────────────────────────────────────│

│──────────────────────────────────────│
│ 합계      333억    667억    +100%    │ ← 합계 행
└──────────────────────────────────────┘
```

**디자인 규칙**:
- **헤더 행**: 다크 그린 배경 + 흰색 텍스트 (Bold, 18pt)
- **홀수 행**: 흰색 배경
- **짝수 행**: 라이트 그레이 배경 (#F5F5F5)
- **합계 행**: 베이지 배경 + Bold
- **테두리**: 외곽 2pt (차콜 그레이), 내부 1pt (라이트 그레이)
- **여백**: 셀 내 상하좌우 5px

#### 정렬 규칙
- **텍스트**: 좌측 정렬
- **숫자**: 우측 정렬
- **퍼센트**: 우측 정렬, 소수점 1자리
- **단위**: 헤더에 표시 (예: "매출(억원)")

#### 강조 표현
- **긍정 값**: 녹색 텍스트 (#28A745) + Bold
- **부정 값**: 빨간색 텍스트 (#DC3545) + Bold
- **경고 행**: 노란색 배경 (#FFF3E0) + ⚠️ 아이콘

- **정렬**: 좌측 정렬

#### 섹션 구분 슬라이드 제목
- **위치**: 중앙 (수평/수직)
- **크기**: 60-72pt, Bold
- **색상**: 다크 그린
- **정렬**: 중앙 정렬

### 7.2 여백 (Margin & Padding)

#### 슬라이드 여백
```
┌────────────────────────────────┐
│ ← 60px                     60px →│
│ ↑ 40px                          │
│                                 │
│    [콘텐츠 영역]                │
│                                 │
│ ↓ 40px                          │
└────────────────────────────────┘
```

- **상단**: 40px (제목 제외)
- **하단**: 40px (출처/페이지 번호 공간)
- **좌우**: 60px

#### 박스 내부 여백
- **패딩**: 20px (상하좌우)
- **박스 간격**: 30px

### 7.3 불릿포인트 스타일

#### Level 1 (주요 항목)
- **기호**: • (검은 원)
- **들여쓰기**: 0px
- **폰트**: 18-20pt, Regular
- **색상**: 차콜 그레이

#### Level 2 (하위 항목)
- **기호**: ◦ (빈 원) 또는 - (하이픈)
- **들여쓰기**: 30px
- **폰트**: 16-18pt, Regular
- **색상**: 차콜 그레이

#### Level 3 (세부 항목)
- **기호**: ▫ (작은 사각형)
- **들여쓰기**: 60px
- **폰트**: 14-16pt, Regular
- **색상**: 라이트 그레이

**예시**:
```
• 주요 전략 (Level 1)
  ◦ 세부 실행 방안 (Level 2)
    ▫ 구체적 액션 (Level 3)
```

### 7.4 강조 표시 방법

#### 텍스트 강조
1. **Bold**: 중요 키워드, 수치
2. **색상 변경**: 녹색(긍정), 빨간색(부정), 파란색(중립)
3. **배경 하이라이트**: 노란색 형광펜 효과 (투명도 30%)
4. **밑줄**: 사용 금지 (하이퍼링크와 혼동)

#### 박스 강조
```
┌─────────────────────────────┐
│ ⚠️ 최우선 과제               │
│                             │
│ 식음팀 인력 충원 (13명)      │
│ → 3개월 내 10명 충원 목표   │
└─────────────────────────────┘
```
- **테두리**: 2pt, 위험 레드 (#DC3545)
- **배경**: 연한 빨간색 (#FFE6E6)
- **아이콘**: 경고 표시 (⚠️)

## 8. 베스트 프랙티스

### 8.1 DO (권장 사항)

#### 콘텐츠
✅ **한 슬라이드 한 메시지**: 슬라이드당 1-2개 핵심 메시지만 전달
✅ **데이터 기반**: 모든 주장은 데이터로 뒷받침
✅ **출처 명시**: 추정치는 "(추정)" 표시, 데이터는 출처 표기
✅ **일관된 용어**: 동일 개념은 동일 용어 사용 (예: "회원권 분양률")

#### 디자인
✅ **여백 확보**: 슬라이드 여백 60px 이상, 박스 간격 30px
✅ **색상 일관성**: 어스톤 팔레트만 사용, 임의 색상 금지
✅ **정렬**: 모든 요소는 그리드에 맞춰 정렬
✅ **폰트 크기**: 최소 16pt 이상 (가독성)

#### 차트
✅ **단순화**: 차트당 최대 5-6개 항목
✅ **라벨 표시**: 모든 데이터 포인트에 라벨
✅ **범례 배치**: 차트 우측 또는 하단
✅ **색상 코딩**: 일관된 색상 규칙 (녹색=긍정, 빨간색=부정)

---

### 8.2 DON'T (피해야 할 사항)

#### 콘텐츠
❌ **텍스트 과다**: 슬라이드당 본문 150자 이하 권장
❌ **글자 크기 축소**: 12pt 이하 사용 금지
❌ **모호한 표현**: "약간", "어느 정도" 등 사용 금지
❌ **데이터 왜곡**: 축 범위 조작, 비율 왜곡 금지

#### 디자인
❌ **과도한 애니메이션**: 애니메이션 최소화 (필요 시 페이드만)
❌ **3D 효과**: 3D 차트, 3D 텍스트 사용 금지
❌ **임의 색상**: 브랜드 팔레트 외 색상 사용 금지
❌ **저해상도 이미지**: 720p 이하 이미지 사용 금지

#### 차트
❌ **파이 차트 7개 이상**: 5-6개 초과 시 막대 차트 사용
❌ **Y축 0 시작 안 함**: 축 범위 조작으로 왜곡 방지
❌ **그리드 과다**: 너무 많은 그리드선은 가독성 저하
❌ **범례 없음**: 모든 차트는 범례 필수
