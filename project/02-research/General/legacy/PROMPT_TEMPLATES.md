  """

  """

  (예: OCC/객실가동률 병기)

## 컨텍스트 제한 대응 패턴
- 3단계 입력 흐름
  - Pass 1 개요: “섹션 개요(헤더·하위목차)만 먼저 제시”
  - Pass 2 본문: “각 하위목차당 2–3단락, 표 자리표시자 포함”
  - Pass 3 근거/표: “표/지표 채우고 출처 병기, 팩트체크 적용”
- 길이 제어: “각 하위목차 120–180단어. 초과 시 요약 블록 추가”
- 누락 데이터: “출처 불명/데이터 없음은 N/A로 명시(가정 금지)”
