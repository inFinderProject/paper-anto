## 슬라이드 제작 시 참고사항

### 전체 톤앤매너
- **전문적(Professional)**: 정확한 용어, 근거 있는 주장
- **간결함(Concise)**: 핵심만 명확히
- **신뢰감(Credible)**: 데이터 기반, 출처 명시
- **객관적(Objective)**: 사실과 분석 구분

### 색상 가이드
- 주요 색상: ANTO 브랜드 컬러 (차분한 네이처 톤)
- 강조 색상: 파란색 (목표/미래), 녹색 (긍정/성장), 주황/빨강 (경고/리스크)
- 배경: 화이트 또는 라이트 그레이

### 폰트 사용
- 제목: 고딕체, Bold, 32-48pt
- 본문: 고딕체, Regular, 16-20pt
- 숫자 강조: Bold, 색상 강조
