  타겟별 서비스 매트릭스

  Primary 타겟 (시간빈곤층) - "40분 리차지" 실현

  핵심 Pain Point: 시간이 없다 → 짧은 시간 내 최대 효과

  | 서비스 영역 | 운영 전략                | SLA   |
  |--------|----------------------|-------|
  | 체크인    | 모바일 사전 체크인, 5분 내 완료  | 5분    |
  | 웰니스    | 90분 압축 패키지 (스파+명상)   | 대기 0분 |
  | F&B    | 사전 주문, 객실 도착 시 세팅 완료 | 10분   |
  | 체크아웃   | 익스프레스 체크아웃           | 3분    |

  시그니처 프로그램: "90분 리차지"
  도착 → 체크인 5분 → 객실 휴식 30분 →
  인피니티풀/스파 45분 → 마무리 10분 → 출발

  Secondary 타겟 (해외VIP) - "Privacy First" 실현

  핵심 Pain Point: 노출되고 싶지 않다 → 완벽한 프라이버시

  | 서비스 영역 | 운영 전략            | 세부 내용           |
  |--------|------------------|-----------------|
  | 프라이버시  | 전용 동선, VIP 엘리베이터 | 로비 미경유 체크인      |
  | 컨시어지   | 전담 버틀러 배정        | 다국어 (영/중/일/아랍어) |
  | 보안     | NDA 서명, CCTV 비공개 | 방문 사실 보안        |
  | 장기체류   | 런드리, 비서 서비스      | 2주+ 장기 패키지      |

  시그니처 프로그램: "Seoul's Secret Stay"
  - 공항 픽업 → 전용 입구 도착 → 버틀러 동행 체크인
  - 객실 내 프라이빗 다이닝, 인룸 스파
  - 외출 시 코디네이터 동행 옵션
