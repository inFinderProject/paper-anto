# 7.4 협업 구조

## 핵심 메시지
> **"2026년 목표 달성을 위해 부서 간 협업 체계를 명확히 정의하고, 신속한 의사결정 구조를 확립한다"**

---

## 협업 구조 개요

### 협업의 3대 원칙

| 원칙 | 설명 | 적용 |
|------|------|------|
| **고객 중심** | 모든 협업은 고객 경험 향상 목표 | Tier별 고객 여정 기준 협업 |
| **책임 명확** | 주담당/협조 역할 구분 | RACI 매트릭스 적용 |
| **신속 대응** | 의사결정 지연 최소화 | 권한 위임 및 에스컬레이션 |

### 협업 구조 프레임워크

```
┌─────────────────────────────────────────────────────────────────┐
│                      경영진 (전략 의사결정)                        │
│              GM · 부장급 · 주간/월간 경영회의                       │
└───────────────────────────────┬─────────────────────────────────┘
                                │
        ┌───────────────────────┼───────────────────────┐
        ▼                       ▼                       ▼
┌───────────────┐       ┌───────────────┐       ┌───────────────┐
│   분양/마케팅   │       │   운영/서비스   │       │   지원/관리    │
│   협업 클러스터  │       │   협업 클러스터  │       │   협업 클러스터  │
├───────────────┤       ├───────────────┤       ├───────────────┤
│ 세일즈팀       │       │ 객실팀        │       │ 재무팀        │
│ 마케팅팀       │       │ 식음팀        │       │ 인사팀        │
│               │       │ 조리팀        │       │ 시설팀        │
└───────────────┘       └───────────────┘       └───────────────┘
```

## 부서 간 협업 매트릭스

### 핵심 프로세스별 협업 관계

#### 1. 회원권 분양 프로세스

```
리드 확보 → 투어 → 상담 → 계약 → 온보딩
```

| 단계 | 주담당 | 협조 | 지원 |
|------|--------|------|------|
| **리드 확보** | 마케팅팀 | 세일즈팀 | - |
| **투어 예약** | 세일즈팀 | 객실팀 | - |
| **현장 투어** | 세일즈팀 | 객실팀, 식음팀 | 시설팀 |
| **상담/제안** | 세일즈팀 | 재무팀 | - |
| **계약 체결** | 세일즈팀 | 재무팀 | - |
| **회원 온보딩** | 객실팀 | 세일즈팀 | 마케팅팀 |

#### 2. 고객 서비스 프로세스

```
예약 → 체크인 → 투숙 → F&B → 체크아웃 → 피드백
```

| 단계 | 주담당 | 협조 | 지원 |
|------|--------|------|------|
| **예약** | 객실팀 | - | 마케팅팀 |
| **체크인** | 객실팀 | 시설팀 | - |
| **객실 서비스** | 객실팀 | 시설팀 | - |
| **F&B 서비스** | 식음팀 | 조리팀 | - |
| **컴플레인 대응** | 객실팀 | 관련 부서 | 경영진 |
| **체크아웃** | 객실팀 | 재무팀 | - |
| **피드백 수집** | 객실팀 | 마케팅팀 | - |

#### 3. MICE/연회 프로세스

```
문의 → 제안 → 계약 → 준비 → 행사 → 정산
```

| 단계 | 주담당 | 협조 | 지원 |
|------|--------|------|------|
| **문의 접수** | 세일즈팀 | - | 마케팅팀 |
| **제안서 작성** | 세일즈팀 | 식음팀, 객실팀 | - |
| **계약 체결** | 세일즈팀 | 재무팀 | - |
| **행사 준비** | 식음팀 | 조리팀, 시설팀 | 객실팀 |
| **행사 진행** | 식음팀 | 조리팀 | 객실팀 |
| **정산** | 재무팀 | 세일즈팀 | - |

## Tier별 협업 체계

### Tier A (전문직/시간빈곤층)

> **핵심 가치**: "효율적 응대, 시간 절약"

| 터치포인트 | 주담당 | 협조 | SLA |
|-----------|--------|------|-----|
| 디지털 리드 | 마케팅팀 | - | 24시간 내 응답 |
| 투어 예약 | 세일즈팀 | 객실팀 | 48시간 내 확정 |
| 체크인 | 객실팀 | - | 5분 이내 |
| 90분 웰니스 | 객실팀 | 식음팀 | 대기 0분 |

**협업 핵심**: 마케팅 → 세일즈 리드 핸드오프 자동화

### Tier B/C (오너/해외 VIP)

> **핵심 가치**: "프라이버시, VIP 프로토콜"

| 터치포인트 | 주담당 | 협조 | SLA |
|-----------|--------|------|-----|
| VIP 문의 | 세일즈팀 | - | 2시간 내 응답 |
| 프라이빗 투어 | 세일즈팀 | 객실팀, 식음팀 | 맞춤 일정 |
| GRO 전담 배정 | 객실팀 | - | 체크인 전 확정 |
| 프라이빗 다이닝 | 식음팀 | 조리팀 | 전용 공간 확보 |
| 버틀러 서비스 | 객실팀 | 전 부서 | 24시간 대응 |

**협업 핵심**: 세일즈 → 객실팀 VIP 정보 사전 공유

### Tier D (법인/MICE)

> **핵심 가치**: "원스톱 서비스, 행사 성공"

| 터치포인트 | 주담당 | 협조 | SLA |
|-----------|--------|------|-----|
| 법인 문의 | 세일즈팀 | - | 24시간 내 응답 |
| 행사 기획 | 세일즈팀 | 식음팀, 객실팀 | 3일 내 제안서 |
| 연회 준비 | 식음팀 | 조리팀, 시설팀 | D-7 최종 확인 |
| 행사 진행 | 식음팀 | 객실팀 | 전담 코디 배정 |
| 사후 관리 | 세일즈팀 | 마케팅팀 | 7일 내 피드백 |

**협업 핵심**: 세일즈 → 식음팀 행사 상세 정보 전달
