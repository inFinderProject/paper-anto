> **아이콘 사용**: 비사용 (텍스트 기반)

## 슬라이드 구성 요약

| 슬라이드 | 제목 | 레이아웃 패턴 | 주요 콘텐츠 |
|:--------:|------|:-------------:|-------------|
| 1 | 섹션 타이틀 | 강조형 | 제목 + 핵심 메시지 |
| 2 | 할인 정책 3대 목적 | 구조형 (3열) | 공공성/수요관리/고객유치 |

- 증가 효과 박스: 녹색 계열 테두리/배경
- 화살표 흐름: 감소 → 증가 → 순효과 시각화
- 결론 박스: 강조 배경, 별표 또는 아이콘 활용
- 핵심 수치: 24pt 이상 대형 폰트, 볼드

## 공통 레이아웃 규격

### 여백 및 간격

```
┌─────────────────────────────────────────┐
│←─ 좌측 여백 40px                        │
│                              우측 40px ─→│
│  ↑                                      │
│  │ 상단 여백 60px (헤더 포함)           │
│  ↓                                      │
│  ┌─────────────────────────────────┐   │
│  │                                 │   │
│  │     콘텐츠 영역                  │   │
│  │     714 × 1003 px               │   │
│  │                                 │   │
│  └─────────────────────────────────┘   │
│  ↑                                      │
│  │ 하단 여백 60px (푸터 포함)           │
│  ↓                                      │
└─────────────────────────────────────────┘
```

### 폰트 규격

- 섹션 제목: 24-28pt, Bold
- 슬라이드 제목: 20-24pt, Bold
- 본문 텍스트: 12-14pt, Regular
- 강조 텍스트: 14-16pt, Bold
- 캡션/주석: 10-11pt, Light

### 색상 가이드

- 주요 강조색: design.css의 --color-primary 참조
- 보조 강조색: design.css의 --color-secondary 참조
- 배경색: 흰색 또는 연한 회색
- 텍스트: 진한 회색 (#333333)
