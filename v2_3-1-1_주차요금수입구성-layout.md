## CSS 클래스 참조

### 레이아웃 컴포넌트

| 클래스 | 용도 | 스타일 |
|--------|------|--------|
| `.slide` | 슬라이드 컨테이너 | 794×1123px, padding 48px |
| `.header-bar` | 상단 헤더 | height 60px, --color-primary |

| `.highlight-box` | 강조 박스 | --color-accent 배경 |
| `.process-flow` | 프로세스 플로우 | flex, 화살표 연결 |
| `.number-badge` | 번호 뱃지 | 원형 32px, --color-primary |
